Visual Hierarchy, Color, and Typography

Visual hierarchy, color, and typography are pivotal elements in web design that significantly impact user experience and the overall effectiveness of a website. Visual hierarchy guides users through content, ensuring that the most important information is prominent and easily accessible, ultimately improving usability. Color choices evoke emotions, set the tone, and help establish brand identity, making it essential to select colors that resonate with the target audience. Typography complements the design by enhancing readability and conveying the site's personality. A well-considered combination of these elements creates a cohesive and aesthetically pleasing design that engages users, communicates effectively, and fosters a memorable and enjoyable online experience.

Class 9 : Responsive site and Files

Responsive Site

In todays lecture we looked at responsive design. We want our website to be able to be used and scale to the type of device we're using. This is tested through Device Emulation - we'd use developer tools to simulate different screen sizes and test designs adaptability.

There was also Cross-Browser Testing: Where we check our designs on different browsers to ensure it is compatible.

Lastly, there was Usability Testing : This is where you involve real users in testing to gather info and feedback on their user experience.

File Oranization

It is important to name your files correctly but before actually starting to name them it's important to plan. We were toldto think of it as a digital workspace.

We looked at types of images - SVG which can be vector, transparent or animated, PNG which can be raster, transparent, animated and lastly, JPG which can be raster and lossy.

Naming consistency is also important when naming files. This helps find and keep them organized.

Writing alt text, or alternative text, is a critical component of web accessibility and plays a signicant role in providing an inclusive user experience. Alt text is a concise textual description of an image, and it serves several essential purposes.
